##  Overview

**Second Web** is a modern, responsive multipurpose website template designed for business, agency, portfolio, and corporate use. It features a clean user interface, responsive layouts, smooth animations, and reusable components, making it suitable for showcasing services, products, or company information.

---

##  Features

* Fully Responsive Design
* Modern and Clean UI
* Bootstrap Layout
* Smooth Scrolling
* Animated Sections (AOS)
* Image Gallery
* Swiper Slider
* Contact Section
* Mobile Friendly
* Cross-Browser Compatible

##  Getting Started

### Clone the repository

```bash
git clone https://github.com/MohamedKhaled31/Second-Web.git
```

### Open the project

```bash
cd Second-Web
```

### Run

Simply open:

```text
index.html
```

Or use **VS Code Live Server** for a better development experience.
